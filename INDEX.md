## ❓ Common Questions

**Q: Do I need coding experience?**
A: No! You can use it as-is. Coding helps for advanced features.

**Q: Can I change the school information?**
A: Yes! Edit `app.py` or `config.py` easily.

**Q: How do I add more FAQs?**
A: Edit `data/faqs.json` following the format.

**Q: Can parents access this from anywhere?**
A: Locally on network only. Deploy online for global access.

**Q: Is it free?**
A: Yes! Free to develop and use locally. Hosting costs vary.

**Q: Can I see what parents asked?**
A: Yes! Check `logs/chat_logs.json`

**Q: How do I make it faster?**
A: Deploy on faster server, or use local caching.

**Q: Is my data safe?**
A: Implement authentication/SSL before using with real data.
